# Security Policy

This project processes sensitive legal text. Please do not report security issues with real personal information or confidential case material.

## Supported Versions

The `main` branch is the only supported branch until the project starts publishing tagged releases.

## Reporting

Open a GitHub security advisory or contact the maintainer privately. Include reproduction steps with synthetic data only.

## Secret Handling

`config/llm.json`, `.env`, generated data, and model artifacts are ignored by git. Keep API keys and private datasets out of commits.
